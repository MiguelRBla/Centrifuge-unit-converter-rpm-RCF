# Centrifuge-unit-converter-rpm-RCF
This is my first repository on GitHub. Contains a centrifuge unit converter made as part of the Fundae Python Essentials 1 course.
The program asks the conversion direction. Then the radius and then the number you want to convert. later delivers the result of the conversion rounded to three decimals.
Afther the conversios it asks if you want to made another one, if negative terminates the program.
